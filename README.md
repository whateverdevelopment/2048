# 2048
[2048.whateverdev.com](https://2048.whateverdev.com/)

WhateverDev's take on the notorious 2048. This project is a fork of *Gabriele Cirulli's 2048* - a game inspired by *Veewo's 1024* and *Saming's 2048*.
